# 计算稿纸 📝

> 像打草稿一样计算 — 支持表达式实时计算、求和、求平均值、百分比计算

### 1. 简介

计算稿纸是一款轻量化桌面计算工具，模拟纸质草稿的使用习惯，让数学计算更自然、高效。

- 保留多行计算历史，可回溯每一步演算过程
- 回车自动带入上一行结果，减少重复输入
- 适配日常计算场景（百分比、千分位数字等）
- 一键复制结果，无缝对接其他办公场景

### 2. 快速开始

* 自动安装（推荐）
  
  * 下载 [不忙脚本盒子](https://www.bm-box.cn) 后，打开 → 脚本市场 → 搜索「计算稿纸」→ 点击安装。
  * 盒子会自动配置 Node.js 环境，安装后通过快捷键或右键菜单即可启动。

* 手动安装
  
  * 需自行安装 Node.js ≥ 16，在脚本目录执行 `node main.js` 启动。

### 3. 启动方式（通过盒子部署）

* 单击启动

* 打开不忙脚本盒子，在已安装脚本列表找到「计算稿纸」，单击即可打开。

* 全局快捷键启动
  
  * 在盒子内为脚本自定义全局快捷键（示例：`Alt + C`），按下快捷键直接唤起独立计算窗口。

* 桌面右键菜单启动
  
  * 桌面/文件夹空白处右键，菜单选择「计算稿纸」快速启动。

### 4. 浏览器兼容说明

工具打开时会调用浏览器窗口，所有浏览器都能正常计算，只是界面外观不同：

- ***Google Chrome / Microsoft Edge（推荐)***  — 能自动隐藏地址栏和标签栏，窗口干净清爽，像独立桌面软件
- ***360 浏览器、QQ 浏览器、Firefox 等其他浏览器*** — 无法隐藏浏览器顶部栏，显示为普通网页，但计算功能完全不受影响

### 5. 全部功能一览

| 功能    | 说明                         |
| ----- | -------------------------- |
| 实时计算  | 输入表达式即时显示结果                |
| 多行稿纸  | 每行独立演算，回车自动带入上一行结果         |
| 百分比运算 | 支持 `100+10%`、`200×5%` 等表达式 |
| 千分位识别 | 自动解析 `1,234.56` 这类带逗号的数字   |
| 快捷复制  | `Ctrl + C` 一键复制当前行结果       |
| 键盘导航  | 上下方向键快速切换各行                |
| 自定义备注 | 为计算结果添加文字标注                |
