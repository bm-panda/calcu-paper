# 计算稿纸

> 像打草稿一样计算 — 支持表达式实时计算、求和、求平均值、百分比计算

## 简介

计算稿纸是一个桌面计算工具，让你像在纸上打草稿一样自然地进行数学计算。支持多行计算历史、回车带入结果、百分比计算、千分位数字处理，还支持 Ctrl+C 一键复制结果。

## 运行环境

本脚本核心依赖 Node.js >= 14 运行环境。推荐用[不忙脚本盒子](https://www.bm-box.cn)运行本脚本，盒子可自动为脚本配置环境依赖，同时原生提供快捷键、右键菜单、定时任务等多种触发方式，开箱即用，大幅降低使用门槛。

## 安装方法

1. 打开**不忙脚本盒子**
2. 进入「脚本管理」
3. 点击「安装脚本」→ 选择 ZIP 包
4. 验证安装结果

## 使用方法

### 快捷键

设置全局快捷键（如 `Alt+C`），按下后自动打开浏览器中的计算稿纸界面。

### 右键菜单

在桌面或文件夹空白处右键，选择「计算稿纸」即可快速打开。

## 浏览器兼容性

本脚本通过浏览器 App 模式（`--app`）打开独立窗口，**仅 Google Chrome 和 Microsoft Edge** 完整支持该特性（可去除地址栏/标签栏，呈现原生桌面应用外观）。

其他浏览器（如 360 浏览器、QQ 浏览器、Firefox 等）无法移除浏览器界面元素，打开后为普通网页窗口，不影响功能使用，仅外观不同。

## 功能特性

- **实时计算**：输入表达式立即显示结果
- **多行稿纸**：每行独立计算，回车自动带入结果到下一行
- **百分比计算**：支持 `100+10%` 等百分比表达式
- **千分位处理**：粘贴带千分位逗号的数字自动识别
- **结果复制**：Ctrl/Command + C 直接复制计算结果
- **方向键导航**：上/下方向键在计算行之间切换
- **备注功能**：为计算结果添加文字备注
